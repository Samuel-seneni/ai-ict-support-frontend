Brainstorm Core Idea: Free AI-Powered ICT Support Ticketing System
Project Name Ideas
•	SchoolHelp AI 
•	Smart ICT Desk 
Core Problem
Many schools and small organisations receive ICT support requests through:
•	WhatsApp messages 
•	Phone calls 
•	Verbal requests 
•	Random emails 
•	Staff walking into IT office 
This causes:
•	Lost requests 
•	Delayed responses 
•	No priority system 
•	No tracking progress 
•	Repeated issues 
•	Frustrated staff and students 
•	Overworked ICT personnel 
Core Solution
Build a free web-based AI-powered ticketing system where users submit ICT issues in one place.
Examples:
•	Printer not working 
•	WiFi down 
•	Password reset 
•	Projector issue 
•	Computer slow 
•	Software installation request 
AI helps by:
•	Categorizing issues automatically 
•	Setting priority (High / Medium / Low) 
•	Suggesting fixes instantly 
•	Routing to correct technician 
•	Creating smart reports 
Main Users
1.	Teachers 
2.	School Admin Staff 
3.	Students 
4.	ICT Technicians 
5.	Small Business Employees 
Main Features
User Side
•	Submit support ticket 
•	Upload screenshot/photo 
•	Track ticket progress 
•	Chat with support 
•	Receive updates by email/SMS 
Admin / ICT Side
•	Dashboard of all tickets 
•	Assign technician 
•	Change status 
•	SLA timers 
•	Reports & analytics 
AI Features
•	Auto classify issue 
•	Auto reply with possible solution 
•	Detect urgent tickets 
•	Predict repeated issues 
•	Knowledge base chatbot 
Goal
To help schools and small organisations manage ICT problems professionally without paying for expensive software.
What Success Looks Like
Short Term
•	5 schools using system 
•	100+ tickets handled 
•	Faster response times 
•	Positive feedback 
Medium Term
•	Used across county schools 
•	Open-source community support 
•	Frontend engineers contributing UI improvements 
Long Term
•	National free support platform for schools 
•	Reduce downtime in education institutions 
•	Youth developers contributing to real impact tech
